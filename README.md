# CUIT课程中心助手
## 介绍
本脚本不鼓励任何形式的学习作弊，不主张懒惰学习行为，仅支持辅助完成额外网络课程，提高学习效率，专业要求课程请务必认真学习。

[成都信息工程大学课程资料仓库](https://github.com/ooyq/cuit-course)

<br>
